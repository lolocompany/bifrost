- go tool staticcheck ./cmd/... ./internal/...

- golangci-lint run ./cmd/... ./internal/...
- go tool errcheck ./cmd/... ./internal/...
- go tool revive -config revive.toml -formatter stylish ./cmd/... ./internal/...

Fix all issues, repeat commands until there are no issues. Summarize all changes made.
